## 🍳 **ဟင်းချက်နည်းလမ်းညွန်း - Cooking Master (အပြည့်အစုံမြန်မာ)**

### ✅ **အဓိကလုပ်ဆောင်ချက်များ**

#### 🥘 **ဟင်းချက်နည်းများ (Recipes)**
- 🍜 ရှမ်းခါးဆွဲ (Shan Noodles) - မြန်မာ
- 🥘 တွမ်ယမ်းဟင်း (Tom Yum Soup) - ထိုင်း
- 🍗 ကုံပေါ်အုံ (Kung Pao Chicken) - တရုတ်
- 🍝 ကာဗုနာရာ (Carbonara) - အီတလီ
- နှင့်နောက်ထပ်များ

#### 🔍 **ရွေးချယ်ခွင့်နှင့်ခွဲခြားမှု**
- 📝 အပြည့်အစုံ ရွေးချယ်ခွင့်
- 🌍 ရန်သုံးအလိုက် ခွဲခြားခြင်း
- ⚡ ခက်ခဲမှုအလိုက် ခွဲခြားခြင်း
- ⏱️ အချိန်အလိုက် ခွဲခြားခြင်း
- 📊 အပြည့်အစုံ အချက်အလက်

#### 🍳 **ဟင်းချက်နည်း အသေးစိတ်**
- 📋 ပစ္စည်းများ စာရင်း
- 📍 လမ်းညွှန်ချက်များ (ဆင့်ဆိုင်း)
- 📊 ကျန်းမာရေးအချက်အလက်
- 💡 ချက်ခြင်းအကြံများ
- ⏱️ အချိန်ခွဲခြားခြင်း

#### ❤️ **အကြိုက်များ**
- ❤️ နှစ်သက်ရသည့်များထည့်သွင်း
- 📌 သိမ်းဆည်းခြင်း
- 🕐 မကြေးမုံးသည့်အခါ
- 📱 လျင်မြန်သောအဆင့်ရ

#### ⏲️ **အချိန်တစ်**
- 🎯 အစီရင်ခံစာလုံးခြင်း
- 📍 အချိန်ခွဲခြားမှု (5/15/30 မိနစ်)
- ▶️ မစ စသည့်ခွင့်
- ⏸️ နားခြင်းခွင့်
- ↻ ပြန်ချက်ခြင်း

#### 🌍 **ဘာသာစကား**
- 🇲🇲 မြန်မာ (MY) - အဓိကဘာသာ
- 🇬🇧 အင်္ဂလိပ် (EN)
- 🇹🇭 ထိုင်း (TH)
- 🔄 တစ်ခြိုက်ထွက်သည့်ဘာသာစကားပြောင်းလဲခြင်း

#### 💾 **ဒေတာသိမ်းဆည်း**
- 🔄 အပြည့်အစုံ မီခေါ်ခြင်း
- 💾 ယာယီသိုက်သိုလှောင်ခြင်း
- 📱 မိုဘိုင်းစက်ပစ္စည်းများ
- 🔐 ကုယ်ပိုင်အချက်အလက်ကာကွယ်ခြင်း

### 🎨 **ဒီဇိုင်းအင်္ဂါ**
- 🎨 လှပသည့်ဒီဇိုင်း
- ✨ အလွယ်တကူမြန်မြန်ရုးရုးခြင်း
- 📱 မိုဘိုင်းကွင်းအညွှန်း
- 🌈 အင်္ဂါများ
- 💡 ထင်ရှားသည့်ကွင်းခွင့်

### 📋 **အသုံးပြုရမည့်နည်း**
1. 🏠 အဟောင်းမုံးခြင်း တစ်ခြေ
2. 🔍 ရွေးချယ်ခွင့်အလိုက် ရွေးချယ်ခြင်း
3. 📖 ဟင်းချက်နည်းအသေးစိတ်ကြည့်ခြင်း
4. ✅ ပစ္စည်းများ ပြည့်ပြည့်သည့်အုံအုံရုံး
5. ⏲️ အချိန်တစ်မှတစ်ဆင့်စီမံခန့်ခွဲခြင်း
